

# System Sequence Diagram （功能模型）

根据我们的[用例分析](https://rookies-sysu.github.io/Dashboard/06-02-use-cases)

## 基本功能

1. 浏览菜单
2. 今日推荐
3. 菜品详细信息
4. 支付订单
5. 搜索菜品
6. 更新菜单

用户可以获取商家菜单，浏览菜品的详细信息，获取商家今日推荐的菜品，手动搜索自己想要的菜品，选择好菜品进行下单
